个人永久免费博客搭建教程

大佬教程：
https://lucyhuafang.blogspot.com/2023/11/hexogithubvercel.html

第一部分：本地部署——把博客搭建在我的电脑，单机版博客。
1.1、下载安装环境node.js
https://nodejs.org/en/

1.2、安装git bash
https://git-scm.com/download/win

1.3、新建一个文件夹，打开，在空白处右键打开git bash。输入第一条命令。
可以复制代码，在git bash空白处右键——paste（粘贴）启用 Hexo镜像
npm config set registry https://registry.npm.taobao.org
再按回车。（后面输入代码都要按Enter）

1.4、在 git bash 中输入第二条命令开始全局安装 Hexo
npm install -g hexo-cli

1.5、在 git bash 中输入第三条
hexo init 自定义博客名
我的博客名是sexyshroud。所以输入hexo init sexyshroud
这时会在文件夹下新建一个以博客名命名的文件夹，博客的本地部署都在这里面。

注意以上这些是打开新建文件夹，再右键打开git bash

1.6、打开新建的sexyshroud文件夹。右键打开git bash
输入hexo g ,编译静态文件到public文件夹，这是根目录会自动生成public文件夹
hexo g

1.7、在 git bash 中输入hexo s，启动本地预览，关闭gitbash后,就无法预览
hexo s

1.8、输入 http://localhost:4000 ，能正常访问，则启动成功
到这里 就是电脑上部署成功，但是单机，要联网还得下一步

第二部分：本地与服务器建立联系
2.1、将hexo代码提交到github仓库，
注册github.com账号
新建一个github仓库，仓库名必须是你的GitHub用户名.github.io，
比如1jia1.github.io 目的是为了搭载 GitHub Pages 服务

2.2、生成SSH密钥

2.3、在 git bash 中输入如下命令

git config –global user.name “你的GitHub用户名”
git config –global user.email “你注册github的邮箱”

2.4、生成密匙命令
ssh-keygen -t rsa -C “你注册github的邮箱”
在本地电脑 C:\Users\fangxu chen中找到新生成的公匙文件id_rsa.pub，记事本格式打开。复制其中一串码（公匙）

2.5、到github网站，点右上角图像，右边找到设置，左边点击SSH and GPG keys。新建一个-New SSH Key。
把刚才复制的乱码粘贴过来。
这一步的作用是让电脑与服务器连接起来。要服务器给我信的电脑授权。
在电脑上生成一个钥匙，把这个钥匙填到服务器，建立信任。

2.6、验证是否连接成功
输入命令
ssh -T git@github.com
会让你输入yes来确认连接
出现Hi sexyshroud! You’ve successfully authenticated, but GitHub does not provide shell access.就表示成功。

2.7、根目录里打开 _config.yml文件，记事本格式打开。找到文件最后一行的 deploy
替换以下代码
deploy:

   type: git

   repo: https://github.com/你的github用户名/你的github用户名.github.io.git,main

（空格不能漏）

第三部分：推送

3.1、安装模块
npm install hexo-deployer-git –save

3.2、清理缓存，重新生成静态文件，推送到github的命令：
hexo clean && hexo g && hexo de
（这条命令在每次本地修改后都要重新打开git bash推送一遍才算完成一次博客更新）
推送成功后，你的博客网站就能联网更新了。进入网站：你的GitHub用户名.github.io

第四部分：管理博客
4.1添加博客
hexo n 博客的名称
创建你的博客就行
再到F:\blog\sexyshroud\source_posts里找到这篇博客的文件夹，用记事本打开。编辑。
最后hexo clean && hexo g && hexo de推送。博客就联网更新了。

https://sexyshroud.github.io/
