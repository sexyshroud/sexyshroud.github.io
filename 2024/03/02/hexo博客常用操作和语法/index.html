hexo博客常用操作和语法

发布新建文章
hexo n 博客的名称
本地博文目录里会多出一个以博文标题命名的文件夹，以记事本方式打开编辑，保存。
文章写好后，直接在git bash中执行以下命令即可直接发布文章
hexo clean && hexo g && hexo de

删除博客
1、删除文件夹source/_posts下目标文章markdown文件
2、删除.deploy_git文件夹
3、执行hexo clean后，再执行hexo g，hexo g 即可。

改变字体颜色
<font color="red"> 希望变成红色的文字 </font>
文本高亮
<mark> 希望高亮的文本 </mark>

<center> 目标元素 </center>
删除线
<s> 删除线文字 </s>
字体
<font face="黑体"> 实现字体为黑体 </font>
<font size=4> 实现字体大小改变 </font>
<big> 比默认字体大一号 </big>
<small> 比默认字体小一号 </small>
